# Specifications

Your product should have 2 components

- **Web service API** (serving RESTful APIs using nodejs / java / python)
- **Front end layer** rendering content (HTML + CSS + JavaScript/TypeScript + toolsets like ReactJS or similar)

## Web Service

Develop a **web service component** that fetches data from the NASA API and exposes a set of simplified endpoints for the UI.

Requirements:

- Cache responses
- Cache expiry + max size
- Follow REST practices
- Runs locally
- Handle API Keys securely

## UI Layer

Ideas:

- Dashboard showing today's APOD
- Date picker to view past APODs
- Gallery view of recent images
- Detailed view for each image (explanation, title, copyright)
- Great design, responsiveness.
